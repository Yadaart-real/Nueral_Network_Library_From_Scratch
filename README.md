Custom Neural Network Framework (From Scratch):
- > Engineered a multi-hidden-layer Artificial Neural Network (ANN) in pure Python/NumPy without high-level ML frameworks.
- > Manually derived and implemented matrix calculus for backpropagation, incorporating He weight initialization and Binary Cross-Entropy loss.
-> Implemented performance and stability techniques including early stopping and gradient updates.

This repository has two files,
1) YKMNueralNet.py -> This script contains the entirety of the Nueral Network Engine/Library
2) YKMnnHandler.py -> This script utilises the custom nueral network library, it handles the input data and training process going into the Nueral Network Engine.
